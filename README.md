# Orbenox ERP Backend

## Overview
Orbenox ERP is a modular backend system designed for inventory, pricing,
product management, and role-based access control.

## Tech Stack
- Java 21
- Spring Boot
- Spring Security (JWT)
- MapStruct
- JPA / Hibernate
- Flyway
- PostgreSQL

## Core Concepts
- Soft delete strategy
- Method-level authorization
- Projection-based reads
- Centralized lookup endpoints
- Localization (i18n)
